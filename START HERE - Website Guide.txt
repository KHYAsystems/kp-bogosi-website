KP Bogosi Empowerment System Website Guide
=========================================

What is inside this folder
--------------------------

1. index.html
   This is the main website page. It contains the actual words, sections, buttons, forms and links.

2. styles.css
   This controls how the website looks: colours, spacing, fonts, mobile layout, cards and animations.

3. script.js
   This controls small interactive features: the mobile menu, scroll animations, stat counters and the booking form WhatsApp redirect.

4. Branding & Concepts
   This folder contains the image files currently used in the website.

Simple meaning of the main website terms
----------------------------------------

Hero section:
The first big section people see when they open the website.

CTA:
Call To Action. This means a button asking visitors to do something, like "Book a Consultation".

Navbar:
The menu at the top of the website.

Footer:
The bottom section of the website with company links and contact details.

Responsive:
The website changes layout to fit phones, tablets and computers.

Mobile-first:
The website is designed to work well on phones first, then it expands for larger screens.

CSS:
The design file. It controls colour, spacing, layout and style.

JavaScript:
The interaction file. It controls things that move or respond when someone clicks.
